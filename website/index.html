hi, this is fuego.

a focus app for macOS. blocks distracting websites and apps so you can focus on what matters. when you try to visit a blocked site, you get a clean focus page instead.

all in native swift. open-source. uses apple's network extensions. free. enjoy.

requirements

macOS 12.0 or later. apple silicon or intel. ~5mb disk space. network extension permission (one-time setup).

get it

download fuego.dmg (latest release)

install via homebrew

enjoy.
